# Slack Messaging Operations

This project demonstrates basic Slack API operations using Node.js.  
It shows how to send, schedule, retrieve, edit, and delete messages using a Slack bot in a safe testing environment.

---

## Features

- **Send a Message** – Post a message to a Slack channel.
- **Schedule a Message** – Schedule messages for future delivery.
- **Retrieve a Message** – Fetch previously sent messages.
- **Edit a Message** – Modify an existing message.
- **Delete a Message** – Remove a message from a channel or conversation.

---

## Prerequisites

- Node.js installed on your system
- A Slack workspace and Slack bot created
- Slack bot token (keep it secret)
